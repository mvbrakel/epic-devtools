# EPIC Devtools

This repository holds tools used in development by team epic.

## Usage

To use the dev tools, include them in your composer.json

    "phing/phing": "enrise/epic-devtools",

## Includes

[] Git PSR-2 pre-commit hook
    [] Check compliance
    [] Fix basics
[]
